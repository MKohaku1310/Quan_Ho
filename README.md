# Quan Ho Bac Ninh

He thong quan ly va gioi thieu dan ca Quan ho Bac Ninh.

## Cong nghe su dung
- Backend: FastAPI
- Frontend: NiceGUI (Python)
- Database: SQLite

## Huong dan chay
1. Cai dat thu vien:
   pip install -r requirements.txt

2. Chay ung dung bang file:
   run.bat

## Thong tin truy cap
- UI: http://localhost:8080
- API: http://localhost:8000/docs
