# He thong Quan Ho Bac Ninh

Ung dung quan ly va gioi thieu dan ca Quan ho Bac Ninh.

## Huong dan chay

1. Cai dat thu vien:
   pip install -r requirements.txt

2. Chay ung dung:
   Click vao file `run.bat`

3. Truy cap:
   Trang chu: http://localhost:8080
   Admin Docs: http://localhost:8000/docs
